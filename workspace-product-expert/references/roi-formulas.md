# 储能 ROI 公式与风控口径

## 户用储能

- 光伏自用提升收益 = 光伏年发电量 * (加储后自用率 - 无储自用率) * (居民购电价 - 余电上网电价)。
- 峰谷套利收益 = 年放电量 * 峰谷价差 * 系统效率。
- 年度总收益 = 保守收益 + 基础收益 + 增值收益。
- 年度净收益 = 年度总收益 - 运维成本 - 保险成本 - 融资成本。
- 静态回收期 = 项目总投资 / 年度净收益。
- NPV = 各年净现金流按折现率折现后求和 - 初始投资。
- IRR = NPV 为 0 时的折现率。

## 工商储

- 单日峰谷套利收益 = 单日放电量 * 峰谷价差 * 系统效率。
- 年峰谷套利收益 = 单日峰谷套利收益 * 年运行天数。
- 需量管理收益 = 最大需量削减量 * 需量单价 * 12。
- 光伏消纳提升收益 = 光伏多自用电量 * (购电价 - 上网电价)。
- 年度净收益 = 峰谷套利 + 需量管理 + 光伏消纳 + 已确认补贴 - 运维/保险/融资/税费。

## 风控规则

- 没有分时电价和负荷数据，不输出工商储正式 IRR 和 NPV。
- 没有光伏发电量、自用率、电价和容量成本，不输出户储正式投资建议。
- 需求响应、虚拟电厂、碳减排、扩容延缓等收益默认归入增值收益，不能混入保守收益。
- 所有 ROI 外发材料必须写明假设、数据来源、敏感性和审核人。
